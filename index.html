Bonjour et bienvenue sur Tolma !

Sur ce site vous trouverez une suite d'énigme concernant quatre aventuriers de l'équipage cosmique Ascheumne :


    • Hélêne Starkerarm
    • Wury Gargarisme
    • Heitor Vonnick
    • Valodia Syna

    • Vous devrez les aider à répondre à différentes énigmes, en éspérant qu'ils s'en sortent tous. Vous n'aurez pas le droit de chercher les réponses dans le code source, ni d'aller les chercher dans une source d'information extérieur. Pour améliorer votre expérience de jeu et empêcher la triche, il est interdit de recharger la page à la moindre broutille.
    Pour vous aider, vous disposerez d'indices si jamais le besoin s'en fait sentir. Pour ne pas vous spoiler durant votre réflexion, nous avons décidé de faire en sorte que ce soit vous qui décidiez du moment où vous obtiendrez ces indices. Nous vous prions donc de ne pas abuser de cette fonction, car notre but est de vous faire réflechir, pas de vous faire perdre votre temps à lire des lignes insipides comme celles-ci.
    Bonne chance, et bon amusement.
